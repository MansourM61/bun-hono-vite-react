# Fullstack Bun-Hono/Vite-React Framework

Hono and Bun as backend, React and Vite as frontend.

An integration of [Hono](https://hono.dev/) based on [Bun](https://hono.dev/docs/getting-started/bun) is used as the backend, while the frontend is powered by [React](https://react.dev/) and [Vite](https://vite.dev/).
